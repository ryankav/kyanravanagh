# Editing the author's own writing

For edits to prose **the author originally wrote** (an essay, draft, or pasted
paragraph). For text the author did *not* originate, see
[`generating-content.md`](generating-content.md).

The author's voice is the product; an edit should make the writing more clearly
*theirs*, not flatten it.

## This is never marked

Editing, correcting, refining, rephrasing, and recommending changes to text the
author originated is **review, not authorship**. The idea and the words are theirs,
so it carries **no `[.ai]` marker** — no matter how much AI helped. Marking tracks
origin, not who touched a sentence last.

## Defaults (until "Author's specifics" overrides them)

- **Propose, don't silently rewrite.** Show the change + a one-line reason.
- **Stay in scope.** Asked for typos → fix typos only. No unrequested rewording or
  reordering.
- **Preserve voice.** Keep idiom, rhythm, punctuation, British spelling, deliberate quirk.
- **Raise structural changes separately**, as a short list, not applied inline.
- If asked to drop in a passage the author did **not** write (AI-generated text kept
  as-is), that is generation, not editing → mark it per
  [`generating-content.md`](generating-content.md).
